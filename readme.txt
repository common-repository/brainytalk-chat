BrainyTalk visa possibilitar uma fácil comunicação entre um cliente e/ou leitor do blog com o proprietário e/ou equipe do mesmo.

== Description ==

BrainyTalk Chat Originalmente foi nomeado como WiseDesk Chat e foi desenvolvido como um projeto open source e adaptado para ser utilizado pelos assinantes do BrainyTalk. O App torna mais fácil a interação entre os visitantes e os administradores do site. Mantendo uma conversa ativa com seus visitantes, os benefícios são enormes, como por exemplo, tirar dúvidas sobre um determinado produto ou serviço na pré-venda, auxiliar seu cliente na escolha de um produto, ou simplesmente conversar e tirar dúvidas de seu cliente podem ser fator definitivo para fechar aquele negócio, ou obter reconhecimento pelo seu trabalho.


== Installation ==

Para o plugin funcionar por completo, basta que o usuário se cadastre no BrainyTalk, em https://web.brainytalk.com e após faça o login através da tela de configuração que estará disponível no painel administrativo do WordPress.


== Frequently Asked Questions ==

= Como faço para acessar o aplicativo web?
Acesse [web.brainytalk.com](https://web.brainytalk.com) e faça seu login

= Como faço para me cadastrar?
Você pode se cadastrar diretamente pela página de administração do plugin, ou acessando [web.brainytalk.com](https://web.brainytalk.com)

= É possível sugerir modificações e ideias para este plugin? =
Sim, seu feedback será sempre bem vindo. Para enviar ideias e ou sugestões utilize o e-mail [talk@brainytalk.com](talk@brainytalk.com)
 
= É necessário pagar para utilizar o plugin? =
O plugin conta com uma versão totalmente gratuita. Mais informações podem ser vistas em [nosso site](https://www.brainytalk.com)
